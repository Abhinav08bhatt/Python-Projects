# Currency Converter

A command-line Python program that allows users to list available currencies, check exchange rates, and convert between currencies using an online currency conversion API.

## Requirements :

- The script must fetch a list of currencies from the API using the requests module.
- Users must be able to list all supported currencies with their names and symbols.
- Users must be able to get the exchange rate between two currencies.
- Users must be able to convert a specific amount from one currency to another.
- The program must continuously run until the user enters q to quit.
- All results must be displayed directly in the terminal.

# Note :
The exchange rates provided by this API are inaccurate and may not match real world currency values.
